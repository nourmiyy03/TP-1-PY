Description

Ce travail consiste à analyser un ensemble de données contenant des notes d’étudiants.

Le programme permet :

La validation et le nettoyage des données

La structuration des informations par étudiant et par groupe

Le calcul des moyennes (avec une fonction récursive)

La détection de certaines anomalies

Le projet a été réalisé en utilisant uniquement les structures de données.
